ABOUT

This game is a passion project by Ceri Woolway, combining a love of the game of Monopoly with a desire to learn more about browser-based coding - mainly Javascript but also HTML, SCSS and CSS - as well as Git/Github. The game is posted here online as a way to showcase my skills.
